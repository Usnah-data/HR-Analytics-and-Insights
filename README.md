# HR-Analytics-and-Insights
This repository contains a comprehensive HR Analytics project where employee data is analyzed to uncover actionable insights for workforce planning and organizational decision-making. The project focuses on identifying trends in employee performance, attrition, satisfaction, and other HR metrics.

Key Features:
Exploratory Data Analysis (EDA) to understand workforce demographics and trends
Visualization of key HR metrics like attrition rate, department performance, and employee satisfaction
Insights to help HR teams improve retention and productivity
Recommendations for data-driven HR strategies

Tools Used:
Excel / Power BI 
Data visualization and dashboard creation
Descriptive and predictive HR analytics

This project showcases my ability to transform HR data into actionable insights, making it an ideal portfolio piece for roles in HR analytics, business analysis, and data-driven decision-making.
